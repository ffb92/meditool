# MediTool – Krankheitsbild-Assistent

Mobile PWA für Pflegekräfte und medizinisches Personal.

## Funktionen

- 15 Krankheitsbilder mit Symptomen, Instrumenten, Materialien, Medikation
- Suchfunktion + Kategorie-Filter
- Offline nutzbar (PWA mit Service Worker)
- Auf allen Geräten installierbar (iOS + Android)

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- Next.js 15 + TypeScript
- Tailwind CSS v4
- next-pwa (Service Worker)
- Vercel Deployment
